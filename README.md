# mindgraph

Simple graphs for Neurosky Mindwave Mobile

You can use this with a [neurosky headset](http://www.amazon.com/NeuroSky-MindWave-Mobile-BrainWave-Starter/dp/B00B8BF4EM) on OSX, I also had to start the "ThinkGearConnector" under Applications/MindWave_Mobile_core/Utilities.

I don't think it really improves upon the bundled app too much, but eventually maybe I will link it with other things and improve the UI. This one is open-source and easy to modify, though!

## development

You can build runtimes with `npm run build`, and you can test (run on current platform) with `npm start`.
